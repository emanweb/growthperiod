# growthperiod WordPress Theme

This repository contains the custom `growthperiod` WordPress theme.

## Local Path

`htdocs/wp-content/themes/growthperiod`

## Git Workflow

1. Make theme changes.
2. Commit locally.
3. Push to GitHub.
4. Deploy files to staging/production.

## Notes

- Theme uses enqueued runtime assets from:
  - `css/main.min.css`
  - `js/main.min.js`
- Keep runtime files in sync when editing source files.
